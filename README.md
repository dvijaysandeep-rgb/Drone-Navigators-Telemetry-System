# Drone Navigator's Telemetry System

## Overview
Drone Navigator's Telemetry System is a UAV-based monitoring and rescue support project designed for real-time object detection and telemetry analysis. The system uses computer vision and AI-based object detection to identify targets and assist in rescue operations.

## Features
- Real-time object detection using YOLOv8
- Live video processing
- Target lock mechanism
- Simulated distance estimation
- Telemetry monitoring dashboard
- Rescue operation support
- FPS monitoring
- UAV surveillance applications

## Technologies Used
- Python
- OpenCV
- YOLOv8
- NumPy
- Computer Vision
- Drone Technology

## Project Workflow
1. Capture live video feed
2. Process frames using YOLOv8
3. Detect objects in real time
4. Calculate object position
5. Estimate distance from target
6. Trigger target lock when object enters center zone
7. Display telemetry information
